# 二项队列

**二项队列**是一系列满足堆性质的树组成的森林。其中的每个树是二项树，每种阶数的二项树最多只有一棵。

其中，$0$ 阶**二项树**是单个节点，而 $k$ 阶**二项树**是由一个 $k-1$ 阶二项树连接到另一 $k-1$ 阶二项树的根节点构成的。

二项树具有以下性质：

- $k$ 阶二项树有 $2^k$ 个节点。
- $k$ 阶二项树是 $k+1$ 叉树。
- 二项树的子树也是二项树。
- $k$ 阶二项树、深度为 $d$ 的一层有 $\binom{k}{d}$ 个节点。

## 合并
合并两个二项队列时，如果都有 $k$ 阶二项树，则将其合并为一棵 $k+1$ 阶二项树。

由于二项队列中的二项树与二进制一一对应，因此其合并与二进制加法一一对应。所以其合并过程与二进制加法器相似。
## 插入
将新插入的节点视作一个单独的二项队列，然后进行合并即可。
## 查找最小值
比较二项队列中的所有二项树的根节点即可，时间复杂度为 $O(\log n)$。
## 删除最小值
在查找最小值的基础上，把该节点删除，并将剩余的节点视作二项队列，与原二项队列进行合并即可。
